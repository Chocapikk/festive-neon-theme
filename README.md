# Custom VSCode Themes

A collection of custom themes for VSCode/Cursor.

## Themes

### Festive Neon

A festive theme with neon red accents, soft colors that are easy on the eyes, and pink strings.

**Features:**
- **Neon Red** (`#ff3366`) : Main color for festive accents
- **Pink** (`#ff99dd`) : For strings
- **Soft Cyan** (`#66ffff`) : For classes and special elements
- **Soft Blue** (`#66b3ff`) : For functions
- **Soft Green** (`#66ff99`) : For positive elements
- **Soft Yellow** (`#ffcc66`) : For constants
- **Dark Background** (`#0f0f15`, `#1a1a25`) : Easy on the eyes

### LeakIX

Based on the LeakIX Hugo theme colors - dark theme with orange/yellow accents.

**Features:**
- **Primary Orange** (`#fab741`) : Main accent color
- **Secondary Orange** (`#ff8c00`) : Secondary accents
- **Dark Background** (`#181f27`, `#090e15`) : Leakix dark theme
- **Green** (`#28a745`) : For strings and success states
- **Cyan** (`#17a2b8`) : For functions and info
- **Yellow** (`#f9b640`) : For warnings and constants

## Installation

1. Copy the folder to `~/.cursor/extensions/custom-vscode-themes/`
2. Reload Cursor
3. Activate via `Ctrl+Shift+P` → "Preferences: Color Theme" → Select "Festive Neon" or "LeakIX"
