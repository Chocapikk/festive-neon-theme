# Festive Neon Theme

A festive theme with neon red accents, soft colors that are easy on the eyes, and pink strings.

## Features

- **Neon Red** (`#ff3366`) : Main color for festive accents
- **Pink** (`#ff99dd`) : For strings
- **Soft Cyan** (`#66ffff`) : For classes and special elements
- **Soft Blue** (`#66b3ff`) : For functions
- **Soft Green** (`#66ff99`) : For positive elements
- **Soft Yellow** (`#ffcc66`) : For constants
- **Dark Background** (`#0f0f15`, `#1a1a25`) : Easy on the eyes

## Installation

1. Copy the folder to `~/.cursor/extensions/festive-neon-theme/` or `~/.vscode/extensions/festive-neon-theme/`
2. Reload Cursor/VSCode
3. Activate via `Ctrl+Shift+P` → "Preferences: Color Theme" → "Festive Neon"
